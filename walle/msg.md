## Msg

弱提示顶（位置顶部，方向向下） 

> extend 底层模块Message

```javascript
new Msg(style, parentId)
```



#### parame

| 参数     | 说明       | 是否必填 | 类型   |
| -------- | ---------- | -------- | ------ |
| style    | 弱提醒样式 | 是       | Object |
| parentId | 父级挂载   | 否       | String |



#### Options

- ##### showMsg: ƒ (content)

  显示弱提示信息，3秒后隐藏
